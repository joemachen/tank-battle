TankBattle — Audio Asset Credits
==================================

All audio assets in this release are procedurally generated placeholders
synthesised using Python standard library only (wave, struct, math, random).
No third-party audio was used. Source code: scripts/generate_audio.py

These files are suitable for development and feel-testing. Before a public
release, replace the WAV files in assets/sounds/ and assets/music/ with
properly licensed recordings. Recommended CC0 / royalty-free sources:

  Recommended sources for replacement assets:
  - https://freemusicarchive.org        (CC-licensed music, filter by CC0)
  - https://opengameart.org             (CC0 game SFX and music)
  - https://incompetech.com             (Kevin MacLeod — CC BY 4.0)
  - https://joshuamclean.com            (synthwave — check individual licenses)
  - https://soundimage.org              (Eric Matyas — free for games)
  - https://freesound.org               (mixed licenses — filter by CC0)
